Readme.txt for embOS for AVR and IAR compiler

Supported hardware:
===================
The sample project Start_ATmega64 was built to run
on the ATmega64 eval board, but may be used
on other target hardware as well.

Configurations / Sessions
=========================

- CSpy_Simulator:
  embOS debug library is used.
  It is prepared for debugging and downloading using the IAR simulator.

- Debug:
  embOS debug library is used.
  It is prepared for debugging and downloading using the Atmel JTAGICE mkII emulator.

- Release:
  embOS release library is used.
  It is prepared for downloading using the Atmel JTAGICE mkII emulator.
